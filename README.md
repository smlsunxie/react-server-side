### base on

[react-boilerplate](https://github.com/koba04/react-boilerplate)

sever side rendering & Flux

ES6 style

-	react
-	react-router
-	express
-	browserify
-	babel

setup
-----

```
npm install
```

watch and autobuild
-------------------

```
npm run watch
```

run server
----------

```
npm start
```

manual build
------------

-	create bundle.js

```
npm run build
```

test (with Jest)
----------------

```
npm test
```
